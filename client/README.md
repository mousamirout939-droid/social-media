# Bloom — frontend (React + Vite)

This is the frontend for Bloom. See the [root README](../README.md) for full
setup and deployment instructions (Vercel).

Quick start:

```bash
npm install
npm run dev
```

Environment variables (see `.env.example`):
- `VITE_API_URL` — backend API base URL (defaults to the Vite dev proxy locally)
- `VITE_SOCKET_URL` — backend base URL for Socket.IO
